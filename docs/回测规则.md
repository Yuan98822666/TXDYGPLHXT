# 游资量化系统研究规范（阶段 1–2）

> 目标定位：
> **以游资思维为核心，量化识别 A 股市场中的风向、情绪与资金合力，
> 在涨停/连板出现之前，提前锁定“具备投机爆发潜力”的板块与个股，
> 服务于“快进快出、赚快钱”的交易目标。**

---

## 一、系统总设计思想（非常重要）

### 1. A 股市场本质假设

* T+1 交易制度
* 涨跌停制度
* 散户主导 + 情绪驱动
* 短期价格 ≠ 基本面，而是 **资金行为 + 市场共识**

**结论：**

> 本系统不做价值投资，不预测长期收益，
> 专注于：
> **“投机情绪如何形成、如何扩散、如何退潮”**

---

### 2. 系统分层原则

| 层级            | 职责       | 特点        |
| ------------- | -------- | --------- |
| Raw（事实层）      | 记录市场原始状态 | 只写不改、不做判断 |
| Behavior（行为层） | 描述市场行为   | 结构、变化、异常  |
| Research（研究层） | 统计验证     | 不直接交易     |
| Strategy（策略层） | 交易规则     | 严格纪律      |

**当前文档覆盖：**

> Raw → Behavior → Research（阶段 1–2）

---

## 二、已完成的 Raw 数据基础（事实层）

### 1. 板块快照表：`raw_block_huoyue`

**作用：**

* 记录某一时间点，一个板块“当下是什么样子”
* 不包含任何判断、不包含选股逻辑

**核心字段分组：**

* 行情：

  * `block_zdf`（涨跌幅）
  * `block_hsl`（换手率）

* 结构：

  * `up_count` / `dw_count` / `pi_count`

* 资金：

  * 主力 / 超大 / 大 / 中 / 小单净流入
  * 各档资金占比

* 时间：

  * `market_time`
  * `kz_no`

---

### 2. 个股快照表：`raw_stock_huoyue`

**作用：**

* 分钟级记录单只股票最原始的行情与资金事实
* 支持事后回放、重算、回测

**核心字段：**

* 行情 / 成交 / 换手 / 市值
* 五档资金流向
* 精确时间戳

---

## 三、研究阶段 1：板块“启动行为”三因子模型

> 研究目标：
> **识别“板块被市场公认之前”的启动阶段**

### 总体原则

* 不预测涨幅
* 不看结果
* 只看“行为是否异常”
* 只与“自身历史”或“横截面”对比

---

## 四、三大启动行为因子（定稿版）

### 因子 A：资金结构先行（Smart Money）

#### 游资含义

> 有“非散户资金”提前入场，但尚未引爆情绪。

#### 核心原始量

```
BIG_MONEY_INFLOW = block_cd_inflow + block_dd_inflow
```

#### 行为特征

1. **连续性**

```
POS_RATIO_A = 最近 N 次快照中 BIG_MONEY_INFLOW > 0 的比例
```

2. **异常强度（Z-Score）**

```
Z_A = (当前 15 分钟窗口均值 - 历史均值) / 历史标准差
```

#### 因子判定（研究期）

```
Factor_A = 1
if POS_RATIO_A ≥ 0.7 and Z_A ≥ 1.5
else 0
```

---

### 因子 B：上涨扩散（Breadth）

#### 游资含义

> 从“个别拉升”向“板块共振”转变。

#### 核心比例

```
UP_RATIO = up_count / (up_count + dw_count + pi_count)
```

#### 行为特征

1. **扩散斜率**

```
SLOPE_B = UP_RATIO(now) - UP_RATIO(15min ago)
```

2. **价格背离**

```
DIVERGENCE_B = SLOPE_B - Δ(block_zdf)
```

#### 因子判定

```
Factor_B = 1
if SLOPE_B 明显上升 and block_zdf 仍较小
```

---

### 因子 C：横截面异动（Attention）

#### 游资含义

> 板块开始“被市场注意到”。

#### 横截面定义（同 kz_no）

```
PCT_C = block_zl_inflow 在全市场板块中的分位数
```

#### 行为特征

1. **跃迁速度**

```
JUMP_C = PCT_C(now) - PCT_C(30min ago)
```

2. **持续性过滤**

```
连续 M 次快照 PCT_C ≥ 90%
```

#### 因子判定

```
Factor_C = 1
if PCT_C ≥ 90% and JUMP_C ≥ 阈值
```

---

## 五、板块启动状态机（核心结构）

```
Block_Activation_Vector = (A, B, C)
```

| 状态 | 向量      | 含义        |
| -- | ------- | --------- |
| 静默 | (0,0,0) | 无关注价值     |
| 潜伏 | (1,0,0) | 资金提前布局    |
| 启动 | (1,1,0) | 最具研究/交易价值 |
| 确认 | (0,1,1) | 市场共识形成    |
| 拥挤 | (0,0,1) | 高风险阶段     |

---

## 六、研究阶段 2：因子 → 结果验证（是否值得做）

### 研究核心问题

> 当某板块 **第一次进入 (A=1,B=1)** 时：
>
> * 后 1–2 个交易日
> * 是否显著更容易产生涨停 / 连板？

---

### 1. 启动事件样本定义

* 一个样本 = 一次“启动事件”
* 触发条件：

  * 某板块首次出现 (A=1,B=1)
  * 前 X 分钟未出现该状态

---

### 2. 结果层指标（只看投机结果）

* R1：是否出现 ≥1 只涨停
* R2：板块内最高连板高度
* R3：是否成为当日 / 次日主线板块

---

### 3. 对照样本（必须）

* 同板块随机时间
* 或同时间弱板块 (A=0,B=0)

---

### 4. 验证标准（最低要求）

```
P(涨停 | 启动期) ≥ 2 × P(涨停 | 对照)
```

否则：

> 模型不具备交易价值，必须修正
