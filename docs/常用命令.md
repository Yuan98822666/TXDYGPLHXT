✅ 创建 venv 

```bash
python -m venv venv
```

✅ 进入 venv
```bash
venv\Scripts\activate
```
✅安装依赖  文件里 一点中文都不能有 简直是活畜生 你妈的  2个小时
```bash
pip install -r requirements.txt
```

```bash
playwright install chromium
```
✅反写依赖
```bash
pip freeze > requirements.tx
```

```bash
python -m scripts.init_db
```
